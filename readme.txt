介绍一批python好用的库

数据可视化的库
Matplotlib/matplotlib.pyplot二维数据可视化
SeaBborn统计类数据可视化功能库
Mayavi三维数据可视化功能库，展示三维数据可视化效果

数据分析库
Numpy
Pandas python数据分析高层次应用库
SciPy 数学、科学和工程计算功能库类似Matlab

文本处理库
PyPDF2 用来处理pdf文件的工具集
NLTK 自然语言处理第三方库，文本分类、标记、句法语法、语义分析等（最优秀的自然语言python库）
Python-docx 创建或更新Microsoft Word文件的第三方库

机器学习库
Scikit-learn 机器学习库
Tensorflow 机器学习框架
MXNet 深度学习计算框架

python网络爬虫
Requests 最友好的网络爬虫功能库
Scrapy 网页爬虫框架
pyspider 强大的web页面爬取系统

python信息解析
Beautiful Soup 提供HTML和XMl的解析库
Re 提供正则表达式解析和处理功能库
Python-Goose 提取文章类型web页面的功能库

python的web网站开发
Django 最流行的web应用框架
Pyramid 提供简单方便构建Web系统的应用框架
Flask Web应用开发微框架

python网络应用开发
WeRoBot 微信公众号开发框架
aip 百度AI开放平台接口
MyQR 二维码生成第三方库


python图像用户界面开发
PyQT5 qt是非常成熟的跨平台桌面应用开发系统，具备完备的GUI，也就是图形界面
wxPython 跨平台GUI开发框架
PyGObject 使用GTK+开发GUI的功能库

python库游戏开发
PyGame 简单的游戏功能开发库
Panda3D 开源、跨平台的3D渲染和游戏开发库
cocos2d 构建2d游戏和图形界面交互式应用的框架

python虚拟现实
VR Zero在树莓派上开发VR应用的python 库
pyvor 针对Oculus Rift设备开发接口
Vizard 基于python的通用VR开发引擎

python图像艺术
Quads 处理图像，迭代
ascii_art ASCII艺术库
turtle 绘图体系
